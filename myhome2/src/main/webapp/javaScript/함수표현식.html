함수 표현식

함수 표현식이란 함수인데 이름이 없는 함수 함수는 처음부터 메모리를 차지하고 있는거와 달리 함수표현식은 메모리를 차지하고 있지 않고 호출될때 비로소 메모리가 확보되고 실행후 메모리 사라짐 보통은 이벤트 핸들러나 콜백함수를 만들때 주로 사용됨 가볍게 만들어서 사용하고 버리는 함수를 말함
선언방법은 변수명 = function (매개변수){ } 함수 이름 없음 왼쪽 변수에 함수의 주소를 전달하고 나중에 변수를 이용해서 함수 호출 콜백함수는 함수를 만드는 사용자와 호출자가 다른 함수를 말함 함수를 만들어서 호출자에게 함수의 주소 전달해주면 호출자는 적당한 시점에 콜백함수 호출 이벤트 핸들러를 구축하거나, 시스템의 일부분만을 변도의 코드로 작성해야 할때 주로 사용 만드는 사람은 전체 시스템의 내부구조를 몰라도 정해진 범위 안에서 함수 만들어 전달해줌
